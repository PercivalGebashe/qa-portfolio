# Web Automation Testing

## Objective
Automate functional and regression tests for sample web applications to demonstrate Selenium and Cucumber expertise.

## Tools & Tech
- Java 17
- Selenium WebDriver
- TestNG / Cucumber
- ExtentReports / Allure
- Maven

## Test Scenarios
- Login with valid/invalid credentials
- Form submission
- Button click and alert handling
- Dropdown and checkbox validation

## Highlights
- Implemented Page Object Model (POM)
- Used Data-Driven Testing (Excel)
- Auto-generated HTML reports
